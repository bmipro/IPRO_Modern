# Legacy Training Workflow Map

Source reviewed: `Training_Videos.zip` supplied July 12, 2026.

The recordings show the former SuxessPoint/IPRO workflow. They are product-behavior references, not visual specifications. New screens should use IPRO branding and the current design system.

## Confirmed workflow areas

- Account and contact management, searching, grouping, editing, printing, and Outlook import
- Account type/category management
- Newsletter and letter creation, editing, recipient selection, sending, and history
- Marketing campaign creation, campaign lists, and targeted campaign sending
- E-card selection and sending
- Custom form creation, form sending, submitted contact forms, and submitted custom forms
- Calendar and appointments
- Website template selection
- Home page editor, custom page manager, About Me editor, and website page lists
- File manager and uploads
- Domain attachment
- Testimonials and approval
- Framed/external link manager
- SEO keyword/meta-tag manager
- Payments and payment history
- Password management
- Dashboard news, sales tips, promotions, and report navigation

## Modernization rules

1. Preserve the user goal and data relationship, not the old page layout.
2. Use IPRO terminology and branding everywhere.
3. Keep reusable content independent from templates.
4. Select recipients at send time for newsletters, campaigns, forms, and other communications.
5. Apply package entitlements at the action boundary and present an upgrade prompt when unavailable.
6. Track sent, delivered, opened, failed, and replied activity where the channel supports it.
7. Expose contextual training from each modern feature after videos are renamed and re-recorded.

## Archive quality notes

- Most usable clips are 640x400 H.264 with MP3 narration and run for roughly one to three minutes.
- Several MP4 files contain audio but no usable video track.
- Several 160-byte MP4 files are failed conversions.
- Three AVI files appear to be duplicate Camtasia/TechSmith source fragments.
- UUID filenames contain no embedded titles, so final feature-to-file mapping requires visual confirmation and descriptive renaming.

## Future training center

Create a Super Admin-managed training catalog with feature code, title, description, role, package, video URL, thumbnail, duration, display order, and active status. Surface the matching tutorial inside each Agent Portal feature without exposing legacy SuxessPoint branding.

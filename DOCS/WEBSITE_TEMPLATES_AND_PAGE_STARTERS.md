# Website Templates And Page Starters

## Choosing A Template

In the Agent Portal, open **My Website**, select a template, and save the website settings.

- Selecting a template applies that template's starter color and presentation.
- The template changes layout, spacing, typography, and navigation presentation.
- Pages, content blocks, menus, domains, and uploaded media remain intact when switching templates.
- After selecting a template, an agent may choose a custom theme color and save it without changing the content.

## Creating A Page

Open **My Website > Pages > New Page**. Enter the page title, URL slug, navigation label, and publishing settings. Then choose a starting preset:

- **Blank page**: one editable text section.
- **About page**: introduction, testimonials, and a call to action.
- **Services page**: hero, services list, and a call to action.
- **Contact page**: hero and contact form.
- **Landing page**: hero, services, testimonials, and a call to action.

The preset creates ordinary content blocks. Agents can edit, move, hide, or delete those blocks afterwards. The same page content is shown on both temporary and connected custom domains.

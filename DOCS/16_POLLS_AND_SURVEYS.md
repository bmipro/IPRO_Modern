# Polls and Surveys

## What It Is

A way to build a short poll (one or a few single-choice questions), send it to your client list the same way a newsletter goes out, and watch responses come in — with a public voting page recipients use to answer, no login required.

## Build a Poll

1. Select **Polls** in the Agent Portal menu (under Marketing).
2. Click **New Poll**.
3. Enter a poll title (internal, for your own reference), an email subject line, and a short intro message shown to recipients.
4. Add one or more questions, each with 2 or more answer options. Use **Add Question** / **Add Option** as needed; every question needs at least 2 options.
5. Click **Save as Draft**.

A poll stays a Draft — fully editable and deletable — until you actually send it.

## Send a Poll

1. From the poll's Preview page, click **Send**.
2. Choose an audience: all newsletter subscribers, clients in a specific account type, or one individual client. This reuses the same subscriber list your newsletters send to.
3. Choose **Send now** or schedule a future date/time (using your agent profile's time zone).
4. Click **Confirm Send**.

Recipients get an email with the poll's intro text and a **Take the poll** button linking to their own personal, one-time voting page.

## View Results

Once a poll has been sent, a **Results** link appears. It shows, per question, how many responses each answer option received and what percentage of the total that represents, plus an overall sent/responded count for the poll.

## Re-voting

Each recipient gets one vote. Reopening the same link after submitting shows a "you've already responded" message instead of the form.

## After Voting

Once a recipient submits (or reopens a link they've already used), they're shown a thank-you message and, if the agent's website is published, automatically taken there a few seconds later — with a "go now" link for anyone who doesn't want to wait.

## Deleting a Poll

Only polls that are still Drafts (never sent) can be deleted. Once a poll has an actual send — even a cancelled one — it's kept permanently so its results stay available.

## Package Availability

Polls and surveys are included with every package. If a package removes access to it later, agents keep any polls already sent and their results, but can't create new polls or reach the review screens until they upgrade.

# Agent Document Library

## What It Is

A private place for an agent to store their own business documents — marketing materials, compliance forms, personal templates — separate from any specific client's record. This is different from **Portal Documents**, which live on an individual client's record and can be shared with that client.

## Upload a Document

1. Select **Documents** in the Agent Portal menu.
2. Choose a file (PDF, Word, Excel, image, TXT, or CSV — max 20 MB per file).
3. Optionally enter a category (e.g. "Marketing", "Compliance") to help organize and filter later.
4. Click **Upload**.

## Find a Document

Use the search box to filter by file name, or the category dropdown to filter by category.

## Download or Delete

- Click **Download** to retrieve a document.
- Click the trash icon to permanently delete a document. This also removes the underlying file from storage.

## Storage Limit

Every package includes this feature, but each package has its own total storage limit:

| Package | Storage limit |
|---|---|
| Silver | 50 MB |
| Gold | 500 MB |
| Platinum | 1000 MB |
| Broker | 1000 MB per user |

The **Documents** page shows a usage bar (e.g. "142 MB of 500 MB used"). If an upload would exceed the limit, it's rejected with a message showing current usage — delete unused documents to free up space, or contact support to discuss more storage.
